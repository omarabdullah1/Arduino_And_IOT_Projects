## üìñ Overview

This repository contains practical implementations of core Arduino and IoT concepts. Each project is designed to demonstrate a specific set of skills, from basic sensor reading and actuator control to cloud connectivity and smartphone integration. The code is well-commented and the circuits are designed for clarity and learning.

## üõ†Ô∏è Technologies & Components Used

### üñ• Microcontrollers
- **Arduino UNO / Nano / Mega** ‚Üí Core controller for robots, vending machines, sensor-based systems, and automation projects.  
- **ESP32 / ESP32-CAM** ‚Üí Smart irrigation system, surveillance car robot, motor control (RemoteXY).  
- **NodeMCU (ESP8266)** ‚Üí Solar panel tracker system and IoT-based applications.  

---

### üì° Sensors & Modules
- **Ultrasonic Sensors (HC-SR04)** ‚Üí Obstacle avoiding robots, parking system, radar project.  
- **IR Sensors** ‚Üí Line follower robots (2, 3, and 4 IR variations).  
- **GPS Module** ‚Üí Arduino V2V system with NRF GPS module.  
- **GSR Sensor** ‚Üí Raw data dumper to Excel + LCD I2C.  
- **Flex Sensors** ‚Üí Flex glove PC mouse control, Parkinson‚Äôs disease arm project.  
- **Moisture Sensor** ‚Üí Soil monitoring and irrigation systems.  
- **Pressure Sensor** ‚Üí Water pressure motor control.  
- **Gas / Air Quality Sensors** ‚Üí Air quality monitoring station.  
- **Proximity Sensor** ‚Üí Massage chair control unit.  
- **Fire Sensor / Flame Detector** ‚Üí Fire alarm system.  
- **Temperature & Weather Sensors (DHT, BMP, etc.)** ‚Üí Weather station Tx/Rx with NRF + OLED.  
- **Color Sensor** ‚Üí Color sorter project.  

---

### ‚öôÔ∏è Actuators & Outputs
- **Motors (DC, Servo, Stepper)** ‚Üí Self-balancing robots, robotic arms, drones, RC airplanes, wire bending machine.  
- **Motor Driver (L298N)** ‚Üí Motor-controlled robots.  
- **Relays** ‚Üí Irrigation pump/motor control.  
- **LEDs & LED Matrix** ‚Üí 8x8 LED Matrix (MAX7219).

  - LCD (I2C) ‚Üí Moisture data dumper, sensor visualizations.

  - OLED ‚Üí Weather station receiver.  
  - Projection Glasses ‚Üí Voltameter project.

- **NRF24L01** ‚Üí  
  - Weather station Tx/Rx  
  - V2V system with GPS  
  - RC airplanes with NRF24L01  
  - Robots with NRF-based control  
- **Bluetooth (HC-05/HC-06)** ‚Üí Robot with BT control, robotic arm with smartphone.  
- **WiFi (ESP8266/ESP32)** ‚Üí ESP32 irrigation with Blynk, ESP32 surveillance car, motor control via RemoteXY.  
- **Processing IDE** ‚Üí Radar project visualization.  
- **RemoteXY** ‚Üí ESP32 motor remote control.

## üöÄ Getting Started

### Prerequisites

* **Arduino IDE:** [Download & Install](https://www.arduino.cc/en/software)
* **ESP8266/ESP32 Board Support:** Install via Arduino IDE Boards Manager.
* **Required Libraries:** Install project-specific libraries (DHT, Blynk, LiquidCrystal\_I2C, etc.) via Arduino IDE Library Manager.

### Installation & Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/omarabdullah1/Arduino_And_IOT_Projects.git
   ```
2. Navigate to the desired project folder.
3. Open the `.ino` file in Arduino IDE.
4. Connect your board (Arduino/NodeMCU/ESP32).
5. Select the correct **board** and **port** in Tools menu.
6. For IoT projects:

   * Configure Blynk/Adafruit IO credentials.
   * Replace `Auth Token` and Wi-Fi credentials in the sketch.
   * Setup dashboard as instructed.
7. Upload the code and monitor using Serial Monitor.
